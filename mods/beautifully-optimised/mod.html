About this Modpack

This is the full description of the Beautifully Optimised modpack. It aims to provide the best possible performance while also including stunning visual enhancements through shaders (Iris) and resource packs.

It includes Sodium, Lithium, Indium, Iris, and a carefully selected set of quality-of-life mods that don't impact performance. Perfect for high-end PCs and potato laptops alike.

Key Features:

  • Optimized with Sodium + Lithium
  • Shader support via Iris
  • Essential QOL mods (e.g., AppleSkin, Mod Menu)
  • Client-side only, server compatible
